# Invalid URL Handler

## Overview
This web application is designed to test the validity of a URL entered by the user. If the URL is invalid, an error message is displayed. If the URL is valid, the application will redirect to that URL.

## Setup
1. Save the HTML file to your local machine.
2. Open the file in a web browser.

## Usage
1. Enter a URL into the text box.
2. Click the 'Submit' button.
3. If the URL is valid, the application will redirect to the entered URL.
4. If the URL is invalid, an error message will display.
